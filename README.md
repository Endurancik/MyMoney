## ✨ Возможности

### 💸 Учёт финансов
- **Быстрый ввод** — напишите `кофе 150` или `зарплата 60000`, программа сама определит тип и категорию
- Доходы и расходы по категориям
- История операций с фильтром по датам
- Редактирование и удаление записей

### 📊 Аналитика и графики
- График доходов/расходов за 30 дней
- Расходы по дням недели
- Сравнение текущего и прошлого месяца
- Прогноз расходов до конца месяца
- Дашборд с общим балансом и топ-категориями

### 🎯 Планирование бюджета
- Лимиты расходов по категориям — предупреждение при превышении
- Финансовые цели с прогресс-баром и дедлайном
- Бюджет на месяц: план vs факт
- Регулярные платежи (аренда, подписки) — добавьте один раз, далее автоматически

### 📋 Дополнительно
- Учёт долгов — кто кому и сколько должен
- Напоминания с уведомлениями
- Экспорт финансового отчёта в **PDF**
- Экспорт и импорт **CSV** для Excel
- Актуальные курсы валют: USD, EUR, GBP, CNY, JPY
- Советы по экономии

## 💾 Данные и безопасность

Все ваши данные хранятся **только на вашем компьютере** в файле `money.db` рядом с программой. Никакие данные никуда не отправляются.

**Перенос на другой компьютер:** скопируйте `MyMoney.exe` + `money.db` — все данные сохранятся.

## ❓ Частые вопросы

**Нужен ли интернет?**
Нет. Только для загрузки курсов валют — это опционально.

**Антивирус ругается на файл?**
Это ложное срабатывание — программа собрана через PyInstaller. Добавьте в исключения антивируса.

**Поддерживается ли macOS / Linux?**
Нет, только Windows 10 и 11.

**Как обновить программу не потеряв данные?**
Замените только `MyMoney.exe`. Файл `money.db` с данными остаётся без изменений.
